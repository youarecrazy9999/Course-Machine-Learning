I. Support Vector Machine Là Gì?¶

  • Nh bit, vi bi ton phn loi nh phn tuyn tnh ta cn v c mt phn tch (vi khng gian 2 chiu th mt phng ny l ng phn tch): $$w^Tx + b = 0$$ phn bit c d liu. Khi du ca hm c lng l $H = \begin{Bmatrix} x \mapsto sgn(w^Tx +b) \, ; w \, \epsilon \, \mathbb{R^{\mathbb{N}}}, b \, \epsilon \, \mathbb{R} \end{Bmatrix}$ s th hin c im d liu x nm cm no ca d liu no.

  • Thng thng s c rt nhiu ng phn tch tha mn c vic ny. Chng ta cn mt ng phn chia sao cho khong cch t im gn nht ca mi class ti ng phn chia l nh nhau, nh th th mi cng bng. Khong cch nh nhau ny c gi l margin (l).

  • Trong SVM, chng ta s gii bi ton i tm ng phn chia sao cho c margin l ln nht. y cng l l do v sao SVM cn c gi l Maximum Margin Classifer.

1.1 Margin¶

  • Margin l khong cch gia siu phng n nhng im d liu gn nht tng ng vi 2 phn lp.
  • SVM c gng ti u thut ton bng cch maximize gi tr margin ny, t tm ra siu phng p nht phn loi 2 lp d liu.

1.2 Support Vectors¶

1.2 Cách tính Margin¶

  • Nh vy, siu phng cn tm c dng :
$$w^{T}x + b = 0$$

Trong :

  • w l weight vector
  • x l input vector
  • b l bias

Vi mi im d liu:

$$y_i\left\{\begin{matrix} +1 \text{ for + samples} & \\-1 \text{ for - samples} \end{matrix}\right.$$
$$y_i(\vec{w}.\vec{x_+} +b) \geqslant 1$$$$y_i(\vec{w}.\vec{x_-} +b) \geqslant 1$$

Ta c th n gin ha thnh phng trnh:

$$y_i(\vec{w}.\vec{x_i} +b) -1 = 0$$
$$Margin = (\vec{x_+} - \vec{x_-}) . \frac{\vec{w}}{\parallel{w}\parallel} \tag{1}$$

Ta c: $$y_i(\vec{w}.\vec{x_i} + b) -1 =0 $$

Vi $y_i=1$: $$(1)(\vec{w}.\vec{x_+} + b) -1 =0 $$

$$\vec{w}.\vec{x_+} = 1- b \tag{2}$$

Vi $y_i=-1$: $$(-1)(\vec{w}.\vec{x_-} + b) -1 =0 $$

$$\vec{w}.\vec{x_-} = - 1- b \tag{3}$$

T (1), (2) v (3), ta c: $$Margin = ((\vec{x_+} . \vec{w}) - (\vec{x_-} . \vec{w})) . \frac{1}{\left \| w \right \|}$$

$$Margin = ((1 - b) - (-1-b)) . \frac{1}{\left \| w \right \|}$$
$$Margin = \frac{2}{\left \| w \right \|}$$

Do , chn c 1 mt phng ti u chng ta phi tm c margin c rng ln nht:

$$max \frac{2}{\left \| w \right \|}$$

T l thun vi:

$$max \frac{1}{\left \| w \right \|}$$

Tng ng vi:

$$min \left \| w \right \|$$

1.3 SVM - Optimization¶

$$\underset{w}{min}\frac{1}{2} {\left \| w \right \|}^2 $$$$subject \, to: \frac{y_i(w^Tx_i+b)}{\left \| w \right \|^{2}} \geqslant \frac{1}{\left \| w \right \|^{2}} ,\ \text{for i=1 ... N}$$

iu kin trn tng ng: $y_i(w^Tx_i+b) \geqslant 1$

  • y, N l s lng cc im d liu $(x_i,y_i)$ cn vic ly bnh phng v chia i nhm d dng tnh ton v ti u li.
  • y l mt bi quadratic optimization chu cc rng buc tuyn tnh v c mt im ti thiu duy nht.

1.4 Soft Margin¶

Đặc vấn đề:¶

Xt 2 trng hp nh hnh:

hnh th nht, SVM vn c c nghim. Tuy nhin, do c mt im d liu mu b nhiu, c khong cch rt gn vi im d liu mu xanh, dn n margin c khong cch rt nh. Nu chp nhn sai st mt lng nh im d liu nh vy, margin s c tng ln v SVM s hot ng tt hn

gii quyt nhng vn nh vy, SVM s dng mt khi nim mi l soft margin, v tn gi k thut ny l soft margin SVM. Trong khi k thut SVM thun cn gi l hard margin SVM

Ta cn xy dng mt hm va ti a margin, va ti thiu cc im d liu sai st

Nm 1995, Vladimir Vapnik gii thiu n slack variable vi cc tnh cht nh sau:

Gii thiu bin "slack":

  • $\xi \ge 0$
  • Vi $0 < \xi \leq 1$ th im d liu nm gia margin v nm trong pha ng ca mt phng. y l 1 margin violation
  • Vi $\xi \geq 1$ im d liu gi l misclassified

Gi $x_{0}$ l im b misclassified, khi th khong cch t $x_{0}$ n "off the street" l: $$\frac{|w^T x_{0} +b|}{\parallel{w}\parallel} + \frac{1}{\parallel{w}\parallel} = \frac{1 + y_{x_{0}}(w^T x_{0} + b)}{\parallel{w}\parallel} = \frac{\xi_{x_{0}}}{\parallel{w}\parallel} \to \xi'$$

Gi $x_{1}$ l im khng b misclassified, nhng nm gia "off the street" v hyperlane, khi khong cch t $x_{1}$ n "off the street" l: $$\frac{1}{\parallel{w}\parallel} - \frac{|w^T x_{1} +b|}{\parallel{w}\parallel} = \frac{1 - y_{x_{1}}(w^T x_{1} + b)}{\parallel{w}\parallel} = \frac{\xi_{x_{1}}}{\parallel{w}\parallel} \to \xi''$$

Hm mc tiu ca soft margin SVM s l: $$\frac{1}{2}\parallel{w}\parallel{}^2 + C (\sum_{i}^{N}\xi_{i})$$

vi iu kin: $\frac{y_i(w^Tx_i+b)}{\parallel{w}\parallel{}} \geqslant \frac{1}{\parallel{w}\parallel{}} - \frac{\xi_{i}}{\parallel{w}\parallel} $. Ngha l: $y_i(w^Tx_i+b) \ge 1 -\xi_{i}$, vi $i=1....N$

Trong , C l hng s dng, c chc nng nh ga tr iu chnh tm quan trng ca margin v vic nh i cc im d liu. C th, khi C nh, gi tr ca hm mc tiu khng b nh hng nhiu, vic ti u vn dnh cho vic ti a margin v ngc li.

def plot_svc_decision_function(model, ax=None, plot_support=True):
    """Plot the decision function for a 2D SVC"""
    if ax is None:
        ax = plt.gca()
    xlim = ax.get_xlim()
    ylim = ax.get_ylim()
    
    # create grid to evaluate model
    x = np.linspace(xlim[0], xlim[1], 30)
    y = np.linspace(ylim[0], ylim[1], 30)
    Y, X = np.meshgrid(y, x)
    xy = np.vstack([X.ravel(), Y.ravel()]).T
    P = model.decision_function(xy).reshape(X.shape)
    
    # plot decision boundary and margins
    ax.contour(X, Y, P, colors='k',
               levels=[-1, 0, 1], alpha=0.5,
               linestyles=['--', '-', '--'])
    
    # plot support vectors
    if plot_support:
        ax.scatter(model.support_vectors_[:, 0],
                   model.support_vectors_[:, 1],
                   s=300, linewidth=1, facecolors='none');
    ax.set_xlim(xlim)
    ax.set_ylim(ylim)
