Adaptive linear neurons¶

Adaline c cng b bi Bernard Widrow v nghin cu sinh Tedd Hoff ca ng y ch sau 1 vi nm so vi thut ton perceptron ca Rosenblatt. Thut ton ny c bit th v bi n minh ha cc khi nim chnh xc nh v gim cc hm chi ph lin tc. iu ny t nn tng cho vic hiu cc thut ton hc my nng cao cho vic phn loi nh: logistic regression, support vector machines, and regression models. S khc bit gia Adaptive v Perceptron l weights c cp nht da trn linear activation function ch khng phi l 1 unit step function nh trong Perceptron. Trong Adaptive th linear activation function (), n gin l nh ngha hn ca net input, vy nn: $$\phi(w^Tx) = w^Tx$$

Trong khi linear activation function dng hc weights th chng ta vn dng threshold function d on im kt thc. Hm ny th ging vi unit step function c cp trc ca Perceptron.

Explain partial derivative?¶

Giảm cost function với gradient descent¶

Mt trong nhng thnh phn chnh ca thut ton hc my c gim st l xc nh objective function v ti u ha trong sut qu trnh hc. Objective function thng l 1 cost function m chng ta mun gim. Vi Adaline chng ta cn nh ngha 1 cost function J, hc weights nh sum of squared errors (SSE) gia kt qu tnh ton v kt qu thc s nhn c. $$J(w)=\frac{1}{2}\sum_i(y^{(i)}-\phi(z^{(i)}))^2$$

Mt tnh cht tuyt vi khc ca cost function ny l n li. Do ta c th s dng 1 thut ton ti u ha n gin gi l gradient descent tm weights v gim cost function phn loi.
Vi hnh mnh ha bn di ta c th thy c tng chnh ca gradient descent l climbing down a hill cho n khi t c gi tr nh nht. Trong mi ln lp chng ta thc hin 1 bc ngc hng vi gradient, y kch thc ca mi bc c xc nh bi gi tr ca learning rate, cng nh dc ca gradient.

Dng gradient descent ta c th update weights bng cch thc hin 1 bc ngc hng vi gradient $\nabla J(w)$, ca cost function $J(w)$. $$w:=w + \Delta w$$ Weiht thay i $\Delta w$ th c nh ngha l m gradient $-\nabla J(w)$ nhn vi learning rate $\eta$. $$\Delta w = -\eta\nabla J(w)$$ Tnh gradient ca cost function chng ta cn tnh o hm tng phn ca cost function i vi mi weights $w_j$. $$\frac{\partial}{\partial w_j} = -\sum_i (y^{(i)} - \phi(z^{(i)}))x^{i}_j$$ V vy c th vit cng thc cp nht ca $w_j$ l: $$\nabla w_j = -\eta\frac{\partial J}{\partial w_j}= \eta\sum_i \big(y^{(i)} - \phi (z^{(i)}) \big)x_j^{(i)}$$ Hn na cp nht weights tnh ton da trn tt c training dataset (thay v cp nht gim weights sau mi ln training mu) l l do ti sao phng php ny cng c gi l batch gradient descent.

class AdalineGD(object):
    def __init__(self, eta=0.01, n_iter=50, random_state=1):
        self.eta = eta
        self.n_iter = n_iter
        self.random_state = random_state
    def fit(self, X, y):
        rgen = np.random.RandomState(self.random_state)
        self.w_ = rgen.normal(loc=0.0, scale=0.01,
        size=1 + X.shape[1])
        self.cost_ = []

        for i in range(self.n_iter):
            net_input = self.net_input(X) # dot prod X (100,2) @ w (2,1) => (100,1)
            output = self.activation(net_input) # output (100,1)
            errors = (y - output) # (100,1)
            self.w_[1:] += self.eta * X.T.dot(errors) # (2,100) @ (100,1) => (2,1) grad vec
            self.w_[0] += self.eta * errors.sum() # ???
            cost = (errors**2).sum() / 2.0
            self.cost_.append(cost)
        return self
    def net_input(self, X):
        return np.dot(X, self.w_[1:]) + self.w_[0]

    def activation(self, X):
        return X

    def predict(self, X):
        return np.where(self.activation(self.net_input(X))>= 0.0, 1, -1)

Cải thiện gradient descent thông qua feature scaling¶

Chng ta s dng phng thc feature scaling c gi l standardization. Cung cp cho d liu ca chng ta tnh cht ca phn phi chun: zero-mean v unit variance. Th tc chun ha ny gip cho gradient descent hi t nhanh hn tuy nhin n khng lm cho tp dataset thnh phn phi chun. Standardization lm thay i gi tr trung bnh ca mi feature sao cho n gia v mi feature c lch chun ca 1 (unit variance).
V d: cho standardize ca feature $j$th, chng ta c th tr trung bnh ca mu $\mu_j$ v chia n vi lch chun $\sigma_j$. $$x^{'}_j=\frac{x_j - \mu_j}{\sigma_j}$$

Mt trong nhng l do ti sao standardization gip cho gradient descent ti u ha hn l tri qua t bc hn tm ra gii php ti u (global cost minimum).

Học máy quy mô lớn và stochastic gradient descent¶

Trong phn trc, chng ta hc lm th no gim cost function bi cc bc ngc hng vi cost gradien t ton b training dataset, l l do ti sao cch tip cn ny cn c gi l batch gradient descent. By gi hy tng tng nu chng ta c b dataset c hng triu data, chy batch gradient descent rt tn km trong cc tnh hung nh th v chng ta cn nh gi li ton b tp d liu mi ln chng ta thc hin 1 bc di chuyn n global gradient minimum.
Thay th ph bin ca batch gradient descent l stochastic gradient descent (SGD) ngoi ra cn c th gi cch cch khc l iterative hoc online gradient descent. Thay v update weights da trn tng li tch ly trn tt c training sample $x^i$. $$\Delta w = \eta \sum_i (y^{(i)} - \phi (z^{(i)}))x^{(i)}$$ Chng ta cp nht weights gim cho mi training example. $$\eta (y^{(i)} - \phi (z^{(i)}))x^{(i)}$$

Mc d SGD xem nh xp x gradient descent. N hi t nhanh hn v thng xuyn cp nht weights, mi gradient th c tnh ton da trn 1 training example. iu ny c li th l SGD c th thot khi local minima nhanh hn nu chng ta lm vic vi nonlinear cost function. t c kt qu tha mn SGD, n th quan trng training dataset hin ti 1 cch random. Chng ti mun xo trn dataset cho mi epoch ngn vng lp.
iu chnh learning rate trong sut qu trnh training:
Trong qu trnh thc hin SGD, fixed learning rate thng c thay th bi adaptive learning rate gim dn theo thi gian.
V d: $$\frac {c_1}{[number of iterations] + c_2}$$ y $c_1$ v $c_2$ l 1 hng s.
Ch :
SGD khng t global minimum m ch rt gn vi n. V dng adaptive learning rating chng ta c th t c cost minimum.

Ci tin khc ca SGD l chng ta c th dng n cho online learning rate. Trong online learning, model ca chng ta c th traning ngay khi d liu mi n. iu ny c bit hu ch khi ta c lng ln d liu tch ly. Dng online learning, h thng c th thch nghi ngay vi nhng thay i v d liu training c th c b i sau khi cp nht model nu kho lu tr c hn.

class AdalineSGD(object):
    def __init__(self, eta=0.01, n_iter=10,
        shuffle=True, random_state=None):
        self.eta = eta
        self.n_iter = n_iter
        self.w_initialized = False
        self.shuffle = shuffle
        self.random_state = random_state
    
    def fit(self, X, y):
        self._initialize_weights(X.shape[1])
        self.cost_ = []
        for i in range(self.n_iter):
            if self.shuffle:
                X, y = self._shuffle(X, y)
            cost = []
            for xi, target in zip(X, y):
                cost.append(self._update_weights(xi, target))
            avg_cost = sum(cost) / len(y)
            self.cost_.append(avg_cost)
        return self
    
    def partial_fit(self, X, y):
        if not self.w_initialized:
            self._initialize_weights(X.shape[1])
        if y.ravel().shape[0] > 1:
            for xi, target in zip(X, y):
                self._update_weights(xi, target)
        else:
            self._update_weights(X, y)
        return self
    
    def _shuffle(self, X, y):
        r = self.rgen.permutation(len(y))
        return X[r], y[r]
    
    def _initialize_weights(self, m):
        self.rgen = np.random.RandomState(self.random_state)
        self.w_ = self.rgen.normal(loc=0.0, scale=0.01, size=1 + m)
        self.w_initialized = True
    
    def _update_weights(self, xi, target):
        output = self.activation(self.net_input(xi))
        error = (target - output)
        self.w_[1:] += self.eta * xi.dot(error)
        self.w_[0] += self.eta * error
        cost = 0.5 * error**2
        return cost

    def net_input(self, X):
        return np.dot(X, self.w_[1:]) + self.w_[0]

    def activation(self, X):
        return X

    def predict(self, X):
        return np.where(self.activation(self.net_input(X)) >= 0.0, 1, -1)
