DECISION TREE¶

Cy quyt nh (Decision Tree) l mt model hp dn nu chng ta quan tm n kh nng din gii. Ging nh tn ca n "Cy quyt nh". Chng ta c th khai ph d liu ca chng ta thng qua mt lot cu hi.
Hy xem xt v d sau, trong chng ti s dng cy quyt nh quyt nh mt hot ng c th trong ngy:

Da vo cc feature ca tp d liu hun luyn, cy quyt nh a ra mt lot cc cu hi tr li nhn ca d liu nh v d trn. Mc d v d minh ha da trn cc bin ri rc nhng trong thc t th d liu lin tc vn c th c chp nhn, ging nh tp d liu Iris.
V d, chng ta c th ct on d liu v sepal width (chiu rng i hoa) tr li cho cu hi nh phn nh: Chiu rng i hoa >= 2.8 cm?

Dng thut ton cy quyt nh, chng ta bt u t gc (root) v phn tch d liu da trn mi feature c kt qu ln nht ca Information Gain(IG). Qu trnh ny lp i lp li cho n khi cc node con l cc node l thun khit. Trong thc t, iu ny c th dn n mt cy rt su vi nhiu nt, d dn n tnh trng tha. V vy chng ta thng mun ct ta cy bng cch t gii hn su cho cy.

Tối đa hóa Information Gain¶

phn tch cc node vi nhiu thng tin nht. Chng ta cn nh ngha mt hm mc tiu m chng ta mun ti u ha thng qua thut ton hc cy. y, hm mc tiu ca chng ta dng ti a ha Information Gain mi ln tch.
Hm mc tiu c th c nh ngha nh sau: $$IG(D_p , f) = I(D_p) - \sum_{j=1}^{m}\frac{N_j}{N_p}I(D_j)$$

Trong :
$f$: l feature thc hin phn chia.
$D_p$ v $D_j$: l tp d liu ca cha v node con th j.
$I$: l thc o tp cht (impurity).
$N_p$: l tng s lng mu training node cha.
$N_j$: l tng s lng mu training node con th j.

Information Gain n gin l sai khc gia tp cht ca node cha so vi tng tp cht ca cc node con. Gim tp cht ca cc node con th Information Gain s cng ln. Tuy nhin n gin v gim khng gian tm kim nn hu ht cc th vin (bao gm c scikit-learn) u thc hin bng cy quyt nh nh phn. C ngha l mi node cha c chia thnh 2 node con. $$IG(D_p , f) = I(D_p) - \frac{N_{left}}{N_p}I(D_{left}) - \frac{N_{right}}{N_p}I(D_{right})$$

Ba cch o tp cht thng dng trong cy nh phn l: Gini impurity ($I_G$), entropy ($I_H$) v classification error ($I_E$). Chng ta bt u vi nh ngha ca entropy cho tt c non-empty class (p(i|t) $\neq$ 0).
Cng thc entropy c nh ngha nh sau: $$I_H(t) = -\sum_{i=1}^{c}p(i|t)log_2p(i|t)$$ Trong :
p(i|t): l xc xut mu thuc v class th $i$ ca node $t$. Entropy s l 0 nu tt c cc mu thuc cng 1 class.

Gini impurity c th c hiu nh l tiu chun gim xc xut phn loi sai.
Cng thc gini impurity c nh ngha nh sau: $$I_G(t) = \sum_{i=1}^{c}p(i|t)(1 - p(i|t)) = 1 - \sum_{i=1}^{c}p(i|t)^2$$

Tuy nhin, trong thc t c gini impurity v entropy u mang li kt qu rt ging nhau. V khng ng ginh nhiu thi gian nh gi cy thay v th nghim cc cch ct ta khc nhau.
Mt bin php o tp cht khc l classification error:
$$I_E(t) = 1 - max\{p(i|t)\}$$

y l mt tiu ch ct ta tuy nhin khng khuyn khch pht trin cy quyt nh. V n t nhy cm hn vi s thay i xc sut class ca cc node. Chng ta c th minh ha iu ny bng 2 kch bn phn tch c th c sau y:

Chng ta bt u vi tp d liu $D_p$ node cha, bao gm 40 mu class 1 v 40 mu class 2. T tp d liu ny chng ta phn tch thnh 2 tp d liu con l $D_{left}$ v $D_{right}$. y chng ta s dng o classification error:

Khi dng classification error th information gain ca c 2 A v B u bng nhau l 0.25.
Dng o gini impurity:

Tuy nhin khi dng o gini impurity s ng h B ($IG_G = 0.1\bar{6}$) hn A ($IG_G = 0.125$).

Tng t entropy cng ng h B hn A:

Lm sao phn tch khong nh phn v value ? Lm sao bit khi no dng o no ? Lm sao chn c su ti u cho cy ?

so snh 3 tiu ch o ta dng 1 biu biu din phm v xc sut [0,1] cho class 1. Lu rng chng ti s thm mt phin bn thu nh ca entropy (entropy / 2) quan st tp cht gini l thc o trung gian gia entropy v li phn loi. M ny nh sau:

import matplotlib.pyplot as plt
import numpy as np
def gini(p):
    return (p)*(1 - (p)) + (1 - p)*(1 - (1-p))
def entropy(p):
    return - p*np.log2(p) - (1 - p)*np.log2((1 - p))
def error(p):
    return 1 - np.max([p, 1 - p])
x = np.arange(0.0, 1.0, 0.01)
ent = [entropy(p) if p != 0 else None for p in x]
sc_ent = [e*0.5 if e else None for e in ent]
err = [error(i) for i in x]
fig = plt.figure()
ax = plt.subplot(111)
for i, lab, ls, c, in zip([ent, sc_ent, gini(x), err], ['Entropy', 'Entropy (scaled)', 'Gini impurity', 'Misclassification error'],
                          ['-', '-', '--', '-.'], ['black', 'lightgray', 'red', 'green', 'cyan']):
    line = ax.plot(x, i, label=lab, linestyle=ls, lw=2, color=c)
ax.legend(loc='upper center', bbox_to_anchor=(0.5, 1.15), ncol=5, fancybox=True, shadow=False)
ax.axhline(y=0.5, linewidth=1, color='k', linestyle='--')
ax.axhline(y=1.0, linewidth=1, color='k', linestyle='--')
plt.ylim([0, 1.1])
plt.xlabel('p(i=1)')
plt.ylabel('impurity index')
plt.show()

from sklearn import datasets
import numpy as np
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.linear_model import Perceptron

iris = datasets.load_iris()
X = iris.data[:, [2, 3]]
y = iris.target
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=1, stratify=y)

sc = StandardScaler()
sc.fit(X_train)
X_train_std = sc.transform(X_train)
X_test_std = sc.transform(X_test)

ppn = Perceptron(eta0=0.1, random_state=1)
ppn.fit(X_train_std, y_train)

from matplotlib.colors import ListedColormap
import matplotlib.pyplot as plt

def plot_decision_regions(X, y, classifier, test_idx=None, resolution=0.02):
    markers = ('s', 'x', 'o', '^', 'v')
    colors = ('red', 'blue', 'lightgreen', 'gray', 'cyan')
    cmap = ListedColormap(colors[:len(np.unique(y))])

    x1_min, x1_max = X[:, 0].min() - 1, X[:, 0].max() + 1
    x2_min, x2_max = X[:, 1].min() - 1, X[:, 1].max() + 1
    xx1, xx2 = np.meshgrid(np.arange(x1_min, x1_max, resolution),
    np.arange(x2_min, x2_max, resolution))
    Z = classifier.predict(np.array([xx1.ravel(), xx2.ravel()]).T)
    Z = Z.reshape(xx1.shape)
    plt.contourf(xx1, xx2, Z, alpha=0.3, cmap=cmap)
    plt.xlim(xx1.min(), xx1.max())
    plt.ylim(xx2.min(), xx2.max())

    for idx, cl in enumerate(np.unique(y)):
        plt.scatter(x=X[y == cl, 0], y=X[y == cl, 1], alpha=0.8, c=colors[idx], marker=markers[idx], label=cl, 
                    edgecolor='black')
    
    if test_idx:
        X_test, y_test = X[test_idx, :], y[test_idx]

        plt.scatter(X_test[:, 0], X_test[:, 1],
        c='', edgecolor='black', alpha=1.0, linewidth=1, marker='o', s=100, label='test set')
    
    X_combined_std = np.vstack((X_train_std, X_test_std))
    y_combined = np.hstack((y_train, y_test))
    plot_decision_regions(X=X_combined_std, y=y_combined, classifier=ppn, test_idx=range(105, 150))
    plt.xlabel('petal length [standardized]')
    plt.ylabel('petal width [standardized]')
    plt.legend(loc='upper left')
    plt.tight_layout()
    plt.show()

Phối hợp đa cây quyết định với random forests¶

Overview¶

Random Forest c th coi nh mt tp hp ca cy quyt nh. tng ng sau l ly trung bnh nhiu cy quyt nh m mi cy u mang tnh high variance. N c th xy dng mt model tng qut hn, t overfitting hn. Random Forest dc v nh l Black box, n d bo tt nhng rt kh gii thch c ch thc s ng sau

Chn mu c thay th v ko thay th: Gi s bn c mt hp cha cc vin bi nh s 1 ti 5

  1. Chn mu c thay th: chn xong bi li b n vo, chn tip: 1,3,3,4,1
  2. Chn mu ko thay th: chn xong bi ko dc b n vo trnh chn li n: 1,3,2,5,4

Các bước thực hiện¶

  1. Rt ra 1 tp con ca training set (bootstrap sample) c n im d liu dc chn ngu nhin (chn mu c thay th )
  2. T tp boostrap sample to decision tree: mi node ngu nhin chn d features (chn mu ko thay th ). Chia node ny ra cc nhnh da trn feature no ti u hm mc tiu nht (VD: c info gain cao nht,...)
  3. Lp li bc 1,2,... k ln
  4. Kt hp kt qu d on t cc cy ny, dng majority vote cho ra kt qu cui cng.

Một vài lưu ý¶

Li th ln ca Random Forest l bn ko phi lo nhiu v vic chn hyperparam tht tt. Bn ch cn quan tm ti s cy trong rng (k), thng l cng nhiu cy cng tt cho vic predict, nhng i li l bn phi tnh nhiu cy (chi ph tnh tng)

Mt bootstrap sample xy 1 decision tree:

  • Nu qu t im d liu (n nh) th cc tree s rt khc nhau. N tng tnh ngu nhin (randomness), gip chng overfitting. Tuy vy, t qu th thng dn ti chuyn model ko dc tng qut lm
  • iu ngc li xy ra cho vic n ln.
  • a s cc trng hp, n c cho l bng vi s records trong training set, thng th n cho ta mt s nh i hp l gia bias-variance.
  • V lng features d, mi ln phn chia, ta mun chn mt s nh hn s lng feature sn c trong training set, scikit-learn mc nh d = m

K-nearest neighbors (KNN)¶

KNN l 1 in hnh ca lazy learner. V khi training, thut ton ny khng hc mt iu g t d liu training.

Thut ton KNN t n kh n gin v c th c tm tt bi cc bc sau:

  1. Chn s K v o khong cch.
  2. Tm K im hng xm gn nht vi im d liu mun phn loi.
  3. Gn nhn cho im d liu theo s phiu nhiu nht

Da trn o khong cch chn, thut ton KNN tm K im trong tp hun luyn c gn nht (ging nht) vi im mun phn loi. Nhn ca im d liu ny s da trn s phiu nhiu nht ca K im gn nht

Ưu điểm¶

  • u im chnh ca cch tip cn t b nh l khi thm d liu training mi th vic phn loi s thch ng ngay ( phc tp ca thut ton khi traning bng 0)
  • Khng cn gi s g v phn phi ca cc class.

Nhược điểm:¶

  • KNN rt nhy cm vi nhiu khi K nh.
  • KNN l mt thut ton m mi tnh ton u nm khu test. Trong vic tnh khong cch ti tng im d liu trong training set s tn rt nhiu thi gian, c bit l vi cc c s d liu c s chiu ln v c nhiu im d liu. Vi K cng ln th phc tp cng s tng ln. Ngoi ra, vic lu ton b d liu trong b nh cng nh hng ti hiu nng ca KNN.
