Perceptron¶

1. Biological neuron¶

  • Dendrites: Nhn thng tin t cc neuron khc
  • Cell nucleus: X l thng tin t Dendrites
  • Axon: c s dng bi cc neurons truyn thng tin
  • Synapse: l s kt ni gia axon v dendrite ca neuron khc

2. Artificial Neuron¶

  • Nm 1943, Warren McCulloch v Walter Pitts a ra khi nim u tin v t bo neuron n gin, c gi l McCulloch-Pitts (MCP) neuron
  • Artificial Neuron l mt hm ton hc da trn m hnh neuron sinh hc, mi neuron nhn cc input v weight u vo, tnh tng ca cc tch input v weight v a qua mt hm activation (nonlinear) a ra output
$$z = w^Tx + b = \sum_{i=1}^m w_{i}x_{i} + b$$$$\hat{y} = f(z)$$
  • w : vector of real-valued weights
  • x : vector of input x values
  • m : total number of samples
  • b : bias
  • f : activation function
  • $\hat{y}$ : output

4. Perceptron Learning Algorithm¶

  • Perceptron c gii thiu bi Frank Rosenblatt nm 1957, ng xut mt Perceptron learning rule da trn MCP neuron
  • Perceptron l thut ton Supervised learning thc hin vic phn loi nh phn
TỔNG QUÁT BÀI TOÁN CỦA PERCEPTRON:¶

  • Cho 2 class c gn nhn, ta tm mt ng thng sao cho ton b im thuc class 1 v 1 pha, ton b im thuc class 2 nm v pha cn li (vi iu kin 2 class l linearly separable).

    • Linear separable l vic tn ti mt ng thng phn chia 2 class

Ý TƯỞNG:¶
  • tng c bn ca PLA l chn ng boundary, xt tng im d liu mt, nu im b misclassified ta s iu chnh sao cho boundary ng vi im , c th thy l nhng im cn li trc classified ng gi li misclassified. Tuy nhin sau mt s bc hu hn, PLA vn s hi t , tc l tm c ng thng phn chia 2 lp, min l 2 lp l linear separable

ĐỊNH NGHĨA:¶

$$\text{net input}: z = w_{1}x_{1} + w_{2}x_{2} + ... + w_{m}x_{m}$$$$\text{unit step function}: \phi(z) = \begin{cases} 1 &\text{if } z \ge \theta \\ -1 &\text{otherwise} \end{cases}$$
  • n gin, ta chuyn $\theta$ sang v tri v t $w_0 = -\theta , x_0 = 1$, khi ta c:
$$\text{net input}: z = w_{0}x_{0} + w_{1}x_{1} + ... + w_{m}x_{m} = W^Tx$$$$\text{unit step function}: \phi(z) = \begin{cases} 1 &\text{if } z \ge 0 \\ -1 &\text{otherwise} \end{cases}$$
  • $w_{0}$ : l bias unit
  • $\phi$ :l unit step function

THE PERCEPTRON LEARNING RULE:¶

  1. Khi to cc weight gn 0.
  2. Duyt ngu nhin qua tng im d liu, $x^{(i)}$:
    1. Tnh gi tr output, $\hat{y}$
    2. Update weights
  3. Khi no khng cn gi tr misclassifed th dng thut ton.
  • Cng thc update weights: $$w_j^{(i)} = w_j^{(i)} + \Delta w_j^{(i)}$$
$$\Delta w_j^{(i)} = \eta(y^{(i)} - \hat{y}^{(i)})x_j^{(i)}$$
  • V d:
    • Nu d on ng: $y^{(i)} = 1$, $\hat{y}^{(i)} = 1$ => $\Delta w_j = \eta(1 - 1)x_j^{(i)} = 0$
    • Nu d on sai: $y^{(i)} = 1$, $\hat{y}^{(i)} = -1$ => $\Delta w_j = \eta(1 - (-1))x_j^{(i)} = 2\eta x_j^{(i)}$

MSP: $w_0 + w_1x_1 + w_2x_2 + ... + w_nx_n = 0$

Proof: $$Z_{new}^{(i)} = (w + 2 \eta x_j^{(i)})^Tx^{(i)} = w^Tx^{(i)} + (2 \eta x_j^{(i)})^Tx^{(i)}$$

import numpy as np

class Perceptron(object):
    def __init__(self, eta = 0.01, n_iter = 50, random_state = 1):
        self.eta = eta
        self.n_iter = n_iter
        self.random_state = random_state
    
    def fit(self, X, y):
        rgen = np.random.RandomState(self.random_state)
        self.w_ = rgen.normal(loc = 0.0, scale = 0.01, size = 1 + X.shape[1]) #loc: mean, scale: standard deviation
        self.errors_ = []
        
        for i in range(self.n_iter):
            errors = 0
            for xi, target in zip(X, y):
                update = self.eta * (target - self.predict(xi))
                self.w_[1:] += update * xi
                self.w_[0] += update
                errors += int(update != 0.0)
#             print('epoch = ', i , '      ', 'loss = ', errors)
            self.errors_.append(errors)
        return self
    
    def net_input(self, X):
        return np.dot(X, self.w_[1:]) + self.w_[0]
    
    def predict(self, X):
        return np.where(self.net_input(X) >= 0.0, 1, -1)

from matplotlib.colors import ListedColormap
def plot_decision_regions(X, y, classifier, resolution =  0.02):
    # setup marker generator and color map
    markers = ('s', 'x', 'o', '^', 'v')
    colors = ('red', 'blue', 'lightgreen', 'gray', 'cyan')
    cmap = ListedColormap(colors[:len(np.unique(y))])
    
    x1_min, x1_max = X[:, 0].min() - 1, X[:, 0].max() + 1
    x2_min, x2_max = X[:, 1].min() - 1, X[:, 1].max() + 1
    xx1, xx2 = np.meshgrid(np.arange(x1_min, x1_max, resolution), np.arange(x2_min, x2_max, resolution))
    Z = classifier.predict(np.array([xx1.ravel(), xx2.ravel()]).T)
    Z = Z.reshape(xx1.shape)
    plt.contourf(xx1, xx2, Z, alpha = 0.3, cmap = cmap)
    plt.xlim(xx1.min(), xx1.max())
    plt.ylim(xx2.min(), xx2.max())

    for idx, cl in enumerate(np.unique(y)):
        plt.scatter(x = X[y == cl, 0], y = X[y == cl, 1], alpha = 0.9, c = colors[idx], marker = markers[idx], label = cl, edgecolor = 'black')
