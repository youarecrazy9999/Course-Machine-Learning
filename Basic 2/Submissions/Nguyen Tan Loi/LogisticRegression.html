Activation function¶

Trong logistic regression, chng ta cn c lng gi tr p - xc sut mt vn xy ra c gi tr trong t 0 -> 1, vi u vo l kt qu ca hm net input c gi tr t $-\infty$ -> $+\infty$. Do chng ta cn mt hm chuyn i gia hai gi tr ny.

Ta c cng thc ca Odds nh sau:

Odds: $\frac{p}{1-p} = \frac{\frac{C_positive}{n}}{\frac{C_nagative}{n}} = \frac{C_positive}{C_nagative}$

Logit: $\ln{\frac{p}{1-p}}$

Hm logit nhn gi tr u vo trong (0, 1) v bin i chng thnh cc gi tr trn ton b phm vi s thc $(-\infty, \infty)$, m chng ta c th s dng biu th mi quan h gia s kt hp tuyn tnh cc feature $z = (w^T.x)$ v xc sut xy ra $(p)$:

$Logit(p(y = 1 |x)) = w^T.x$

$\Leftrightarrow \ln{\frac{p}{1-p}} = z$ $\Leftrightarrow \frac{p}{1 - p} = e^{z} $

$\Leftrightarrow p = e^z(1 - p) $ $\Leftrightarrow p = e^z - e^zp $

$\Leftrightarrow e^z = p + e^zp $ $\Leftrightarrow e^z = p(1 + e^z) $

$\Leftrightarrow p = \frac{e^z}{1 + e^z} = \frac{1}{e^{-z}(1 + e^z)} = \frac{1}{1 + e^{-z}}$

Nh vy ta t $\phi{(z)} = \frac{1}{1 + e^{-z}}$ l activation function ca Logistis regression

Lost Function¶

Logistic Regression s dng maximize likelihood c tnh c w.

Nu nh MSE l phng php c s dng c tnh cc tham s trong Adaline th Logistic regression s dng maximun likelihood c tnh tham s w.

  • Xt N mu d liu c gn nhn 0 hoc 1:
    • Vi mu d liu c gn nhn 1: chng ta mun c tnh tham s W sao cho gi tr u ra $\phi{(z)}$ gn vi xc sut l 1 nht c th.
    • Vi mu d liu c gn nhn 0: chng ta mun c tnh tham s W sao cho gi tr u ra $1 - \phi{(z)}$ gn vi xc sut l 1 nht c th.
  • Tm li:
    • Vi tt c mu c nhn 1, chng ta mun c tnh $\hat{y}$ sao cho tch ca tt c xc sut c in kiu ca cc mu ny sao cho gn 1 nht c th. Ngha l cn maximize $\prod_{ i \in \{y = 1\}} \phi{(z^{(i)})}$ (1)
    • Tng t vi tt c mu c nhn 0, ta cn maximize $\prod_{ i \in \{y = 0\}} 1 - \phi{(z^{(i)})}$ (2)
  • Kt hp (1) v (2) ta mun c tnh W sao cho maximize hm $L(w) = \prod_{ i \in \{y = 1\}} \phi{(z^{(i)})} \times \prod_{ i \in \{y = 0\}} 1 - \phi{(z^{(i)})}$ cho ton b dataset.

$L(w) = \prod_{ i \in \{y = 1\}} \phi{(z^{(i)})} \times \prod_{ i \in \{y = 0\}} 1 - \phi{(z^{(i)})} = \prod_i^n {\phi{(z^{(i)})}}^{y^{(i)}} \times (1 - \phi{(z^{(i)})}) ^{1 - y^{(i)}} $ gi l likelihood function

Trong thc th, maximize logarit c s e ca hm ny d dng hn.

log-likelihood function: Ta c:

$$l(w) = \log{L(w)} = log({\phi{(z^{(i)})}}^{y^{(i)}} \times (1 - \phi{(z^{(i)})}) ^{1 - y^{(i)}}) $$$$= log({\phi{(z^{(i)})}}^{y^{(i)}}) \times \log((1 - \phi{(z^{(i)})}) ^{1 - y^{(i)}})$$$$= y^{(i)}\log{({\phi{(z^{(i)})}})} + (1 - y^{(i)})\log{(1 - \phi{(z^{(i)})})}$$

V vy, suy ra c:

$l(w) = \log{L(w)} = \sum_{i = 1}^n [ y^{(i)}\log{({\phi{(z^{(i)})}})} + (1 - y^{(i)})\log{(1 - \phi{(z^{(i)})})}]$

  • S dng log gim thiu kh nng xy ra li numberical underflow, c th xy ra khi hm likelihood qu nh $~ 10^{-123}$.
  • Log cn dng chuyn i du $\times$ thnh $+$ iu ny lm cho vic o hm d dng hn.

By gi c th s dng GD maximize log-likelihood function hay minimize n khi vit di dng cost function nh chapter 2: $$J(W) = \sum_{i = 1}^n [- y^{(i)}\log{({\phi{(z^{(i)})}})} - (1 - y^{(i)})\log{(1 - \phi{(z^{(i)})})}]$$

Cost function mt mu d liu: $$J(\phi(z),y;w) = - y\log{({\phi{(z)}})} - (1 - y)\log{(1 - \phi{(z)})}$$

Nhn vo cng thc trn, c th thy rng:

$$j(\phi(z),y;w) = \begin{cases} -\log(\phi(z)) & if &y =1 \\ -\log(1- \phi(z)) & if & y =0 \end{cases}$$

Tính toán GD trongcost function¶

tnh ton GD trong cost function ca Logistic Regression, ta o hm ring ca hm chi ph theo w:

$\frac{\partial{i(w)}}{\partial{w}} = -\frac{\partial}{\partial{w}}[ylog(\phi(z)) + (1 - y)log(1 - \phi(z)) ]$

$= -(y\frac{1}{\phi(z)} - (1 - y)\frac{1}{\phi(z)})\frac{\partial}{\partial{w}}\phi(z) $ $(1)$

Ta c: $\frac{\partial}{\partial{w}}\phi(z) = \frac{\partial}{\partial{w}}\frac{1}{1 + e^{-z}}$

$= \frac{e^{-z}}{(1+e^{-z})^2}\frac{\partial}{\partial{w}}z$$=$$\frac{1}{1+e^{-z}} \frac{e^{-z}}{1+e^{-z}} \frac{\partial}{\partial{w}}z $

$= \phi(z)(1 - \frac{1}{1+e^{-z}})\frac{\partial}{\partial{w}}z = \phi(z)(1 - \phi(z))x$$(2)$

Thay (2) vo (1), ta c:

$\frac{\partial{j(w)}}{\partial{w}} = -(y\frac{1}{\phi(z)} - (1-y)\frac{1}{1-\phi(z)} \phi(z)(1 - \phi(z) )x$

$= -[y(1-\phi(z)-(1-y)\phi(z)]x = (y-y\phi(z) - \phi(z) + y\phi(z))x$

$= -(y - \phi(z))x$

T suy ra o hm ring ca ton dataset l:

$\frac{\partial}{\partial{W}} = -\sum_{i=1}^n(y^{(i)}-\phi(z^{(i)}))x^{(i)} = \nabla J(W)$

S dng GD, chng ta c th update W theo cch i ngc chiu gradient, $\nabla J(W)$, ca cost function $J(W)$:

$$W = W + \Delta W$$

Vi $\Delta W = -\eta \nabla J(W) = -\eta[-\sum_{i=1}^n(y^{(i)}-\phi(z^{(i)}) )]$

$= \eta \sum_{i=1}^n(y^{(i)}-\phi(z^{(i)}))$

class LogisticRegressionGD(object):
    """Logistic Regression Classifier using gradient descent.

    Parameters
    ------------
    eta : float
      Learning rate (between 0.0 and 1.0)
    n_iter : int
      Passes over the training dataset.
    random_state : int
      Random number generator seed for random weight
      initialization.


    Attributes
    -----------
    w_ : 1d-array
      Weights after fitting.
    cost_ : list
      Logistic cost function value in each epoch.

    """
    def __init__(self, eta=0.05, n_iter=100, random_state=1):
        self.eta = eta
        self.n_iter = n_iter
        self.random_state = random_state

    def fit(self, X, y):
        """ Fit training data.

        Parameters
        ----------
        X : {array-like}, shape = [n_examples, n_features]
          Training vectors, where n_examples is the number of examples and
          n_features is the number of features.
        y : array-like, shape = [n_examples]
          Target values.

        Returns
        -------
        self : object

        """
        rgen = np.random.RandomState(self.random_state)
        self.w_ = rgen.normal(loc=0.0, scale=0.01, size=1 + X.shape[1])
        self.cost_ = []

        for i in range(self.n_iter):
            net_input = self.net_input(X)
            output = self.activation(net_input)
            errors = (y - output)
            self.w_[1:] += self.eta * X.T.dot(errors)
            self.w_[0] += self.eta * errors.sum()
            
            # note that we compute the logistic `cost` now
            # instead of the sum of squared errors cost
            cost = -y.dot(np.log(output)) - ((1 - y).dot(np.log(1 - output)))
            self.cost_.append(cost)
        return self
    
    def net_input(self, X):
        """Calculate net input"""
        return np.dot(X, self.w_[1:]) + self.w_[0]

    def activation(self, z):
        """Compute logistic sigmoid activation"""
        return 1. / (1. + np.exp(-np.clip(z, -250, 250)))

    def predict(self, X):
        """Return class label after unit step"""
        return np.where(self.net_input(X) >= 0.0, 1, 0)
        # equivalent to:
        # return np.where(self.activation(self.net_input(X)) >= 0.5, 1, 0)

from matplotlib.colors import ListedColormap
import matplotlib.pyplot as plt


def plot_decision_regions(X, y, classifier, test_idx=None, resolution=0.02):

    # setup marker generator and color map
    markers = ('s', 'x', 'o', '^', 'v')
    colors = ('red', 'blue', 'lightgreen', 'gray', 'cyan')
    cmap = ListedColormap(colors[:len(np.unique(y))])

    # plot the decision surface
    x1_min, x1_max = X[:, 0].min() - 1, X[:, 0].max() + 1
    x2_min, x2_max = X[:, 1].min() - 1, X[:, 1].max() + 1
    xx1, xx2 = np.meshgrid(np.arange(x1_min, x1_max, resolution),
                           np.arange(x2_min, x2_max, resolution))
    Z = classifier.predict(np.array([xx1.ravel(), xx2.ravel()]).T)
    Z = Z.reshape(xx1.shape)
    plt.contourf(xx1, xx2, Z, alpha=0.3, cmap=cmap)
    plt.xlim(xx1.min(), xx1.max())
    plt.ylim(xx2.min(), xx2.max())

    for idx, cl in enumerate(np.unique(y)):
        plt.scatter(x=X[y == cl, 0], 
                    y=X[y == cl, 1],
                    alpha=0.8, 
                    c=colors[idx],
                    marker=markers[idx], 
                    label=cl, 
                    edgecolor='black')

    # highlight test examples
    if test_idx:
        # plot all examples
        X_test, y_test = X[test_idx, :], y[test_idx]

        plt.scatter(X_test[:, 0],
                    X_test[:, 1],
                    c='',
                    edgecolor='black',
                    alpha=1.0,
                    linewidth=1,
                    marker='o',
                    s=100, 
                    label='test set')

Xử lý overfitting bằng phương pháp Regularization¶

Khi mt model a kt qu training qu tt (d liu o to) nhng khi a vo d liu test li khng khi qut tt.
L do:

  • M hnh c phng sai qu cao
  • C qu nhiu tham s khi train
    $\Rightarrow $ M hnh qu phc tp vi d liu hin ti
  • Do d liu train b thiu
    $\Rightarrow $ M hnh khng phc tp nm bt tt mu d liu train $\Rightarrow $ Khng th hin c d liu test (model c hiu sut thp)

Mc d chng ta ch mi s dng cc model tuyn tnh phn loi d liu, cc vn v qu mc v thiu cn bng c th c minh ha bng cch so snh ranh gii quyt nh tuyn tnh vi cc ranh gii phi tuyn phc tp hn:

Thng thng, cc nh nghin cu s dng thut ng $bias$ v phng sai $variance$ hoc $bias-variance tradeoff$ m t hiu sut ca mt model
C ngha l: nu mt im d liu b lch kh nhiu so vi cc im cng loi
$\Rightarrow$ lch cao $\Rightarrow$ phng sai cao t l thun vi mc qu cao v mc qu cao t l thun vi mc thiu.

Do , ta c th ni rng model c 1 $high$ $variance$ hoc 1 $high$ $bias$

Mt cch tm 1 'good bias-variance trade-off' l iu chnh phc tp ca model thng qua chnh quy ha (regularization).
Regularization l 1 phng php rt hu ch x l cng tuyn (tng quan cao gia cc feature), lc d liu b nhiu v cui cng l ngn chn overfitting.

Regularization thm 1 thng tin (bias) 'trng pht' cc gi tr tham s cc tr (trng s).
Hnh thc chnh quy ha ph bin nht gi l chnh quy L2 (L2 shrinkage hay weight decay), cng thc c vit nh sau:

$$\frac{\lambda}{2}||w||^2 = \frac{\lambda}{2}\sum_{j=1}^mw_j^2$$

$\lambda$ c gi l tham s chnh quy ha (regularization parameter)

Thm cng thc ny vo Cost function ca Logistic Regression c th gim bt trng s trong qu trnh o to model:

$$J(w) = \sum_{i=1}^n[-y^{(i)}log(\phi(z^{(i)})) - (1 - y^{(i)})log(1 - \phi(z^{(i)}))] + \frac{\lambda}{2}||w||^2$$

Trong $\lambda$ gi l tham s chnh quy, $ \lambda = \frac{1}{C}$
